# Pinwheel 扩展使用指南

欢迎使用 Pinwheel 浏览器扩展！这款工具可以帮助您高效管理浏览器中的标签、书签和历史记录，并支持本地定制化短链功能。

## 快速开始

### 安装扩展

从浏览器应用商店下载并安装 Pinwheel 扩展，安装完成后，您将在浏览器工具栏看到 Pinwheel 图标。

### 首次使用设置

1. 点击浏览器工具栏中的 Pinwheel 图标
2. 按照提示完成初始设置
3. 授予必要的权限以访问您的标签、书签和历史记录

## 核心功能

### 🔍 一体化搜索

Pinwheel 提供了强大的搜索功能，可以同时在您的标签、书签和历史记录中查找内容。

```js
// 搜索功能示例代码
function searchResources(query) {
  const tabs = searchInTabs(query);
  const bookmarks = searchInBookmarks(query);
  const history = searchInHistory(query);
  
  return {
    tabs,
    bookmarks,
    history
  };
}
```

### 🏷️ 智能标签管理

自动整理和分类您的浏览器标签，让您的工作空间更加整洁有序。

| 功能特性 | 描述 |
|---------|------|
| 标签分组 | 自动识别相关标签并创建分组 |
| 标签休眠 | 自动休眠不常用的标签以节省资源 |
| 标签快照 | 保存当前标签页配置以便稍后恢复 |

## 使用技巧

### 键盘快捷键

使用以下键盘快捷键可以提高您的工作效率：

- `Alt+P`：打开 Pinwheel 主界面
- `Alt+K`：快速搜索
- `Alt+L`：管理短链

## 常见问题

### 为什么我看不到某些历史记录？

Pinwheel 只能访问浏览器授予权限的历史记录。请确保您已在浏览器设置中允许 Pinwheel 访问您的完整浏览历史。

### 短链会同步到云端吗？

不会，所有短链都仅存储在您的本地设备上，确保您的隐私安全。
