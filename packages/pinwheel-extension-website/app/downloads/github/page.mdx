# 通过 GitHub 安装 Pinwheel 扩展

如果您无法访问 Chrome 应用商店，可以通过 GitHub Release 下载 Pinwheel 扩展并在开发者模式下运行。

## 下载扩展文件

1. 访问 [Pinwheel 的 GitHub Release 页面](https://github.com/levenx/pinwheel/releases)
2. 下载最新版本的扩展打包文件（通常是 .zip 格式）

## 安装步骤

### 1. 解压文件
将下载的 .zip 文件解压到您选择的文件夹中。

### 2. 打开扩展管理页面
在 Chrome 浏览器中输入 `chrome://extensions/` 并按 Enter。

### 3. 开启开发者模式
在扩展管理页面右上角，打开 "开发者模式" 开关。

### 4. 加载扩展
点击 "加载已解压的扩展程序" 按钮，选择您解压的文件夹。

### 5. 完成安装
安装后，您会在扩展列表和浏览器工具栏中看到 Pinwheel 图标。

## 常见问题

- **扩展无法加载**：确保下载最新版本，检查解压完整性（应有 manifest.json 文件）
- **未打包提示**：这是正常现象，点击 "继续" 或 "保留" 即可
- **更新扩展**：下载新版本并重复上述安装步骤

如有问题，请访问 [GitHub Issues](https://github.com/levenx/pinwheel/issues) 寻求帮助。
