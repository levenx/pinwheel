---
title: 通过 Gitee 安装 Pinwheel 扩展
---

### 无法访问 Chrome 插件商店？没关系！

对于无法访问外网的用户，我们提供了通过 Gitee 平台下载和安装 Pinwheel 扩展的方法。请按照以下步骤操作：

## 下载扩展文件

1. 访问 Pinwheel 的 Gitee Release 页面：
   [https://gitee.com/levenx/pinwheel/releases](https://gitee.com/levenx/pinwheel/releases)

2. 在最新版本的发布信息中，找到并下载扩展打包文件（通常为 `.zip` 格式）

## 安装扩展到 Chrome 浏览器

1. 打开 Chrome 浏览器，点击右上角的三个点（设置菜单）

2. 选择 **更多工具** > **扩展程序**

3. 在扩展程序页面右上角，开启 **开发者模式** 开关

4. 点击页面左上角的 **加载已解压的扩展程序** 按钮

5. 选择你刚才下载并解压的扩展文件夹

6. Pinwheel 扩展将立即安装到你的浏览器中

## 常见问题

### 扩展无法加载或出现错误
- 确保下载的是最新版本的扩展文件
- 检查解压是否完整，文件夹结构是否正确
- 尝试重新启动 Chrome 浏览器后再次安装

### 安装后扩展功能不正常
- 确保使用的是最新版本的 Chrome 浏览器
- 确认扩展已经获得了所需的权限
- 可以尝试卸载后重新安装
